--- COMMENTS ----
Number of frequencies 
FLAG Current density STT:
    0: Off
    1: On

FLAG Current density STT type:
    DC - AC - Pulse: 0 off, 1 on 

FLAG Current Frequency STT:
    0: 1 frequency. Only 'Current frequency STT' used.
    1: Range of frequencies. 'Minimum', 'Maximum' and 'Number of frequencies' used.

FLAG Spin polarization function:
    0: g_T = P
    1: g_T = 2P/(1+(P^2)mxp)

FLAG field-like torque STT:
    0: Off
    1: On (FLAG Current density STT must be activated)

FLAG Current density SOT:
    0: Off
    1: On

FLAG Current density SOT type:
    DC - AC - Pulse: 0 off, 1 on

FLAG Current Frequency SOT:
    0: 1 frequency. Only 'Current frequency SOT' used.
    1: Range of frequencies. 'Minimum', 'Maximum' and 'Number of frequencies' used.

FLAG field-like torque SOT:
    0: Off
    1: On (FLAG Current density SOT must be activated)

FLAG External field:
    0: Off
    1: On

FLAG External field type:
    DC - AC - Pulse: 0 off, 1 on

FLAG external field frequency:
    0: 1 frequency. Only 'External field frequency' used.
    1: Range of frequencies. 'Minimum', 'Maximum' and 'Number of frequencies' used.

FLAG VCMA: 
    0: Off
    1: On

FLAG VCMA type:
    DC - AC - Pulse free layer - Pulse polarizer layer: 0 off, 1 on

FLAG VCMA field frequency:
    0: 1 frequency. Only 'VCMA Frequency' used.
    1: Range of frequencies. 'Minimum', 'Maximum' and 'Number of frequencies' used.
    2: multiple of current frequency/frequencies

FLAG Thermal field:
    0: Off
    1: On

** file Pulse:	     The value of current J changes over the time. The first column of the file contains the times to which a change of J corresponds; 
		     the second column contain the value of J at the corresponding time. 
		     The value between two J is chosen with a linear interpolation. If you want a pulse, you have to set different value of J in consecutive times.
		     Same for VCMA and external field
